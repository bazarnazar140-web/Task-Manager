# Task Manager (Менеджер задач)

## Опис проєкту

«Менеджер задач» — це консольний застосунок, розроблений мовою програмування C++ у рамках навчальної практики зі спеціальності 121 «Інженерія програмного забезпечення».

Програма призначена для створення та керування списком задач користувача.

## Основні можливості

* додавання нової задачі;
* встановлення пріоритету задачі;
* перегляд усіх задач;
* перегляд лише активних задач;
* позначення задачі виконаною;
* видалення задач;
* збереження даних у файл;
* завантаження даних після повторного запуску програми;
* обробка некоректного вводу.

## Використані технології

* C++;
* STL (Standard Template Library);
* файловий ввід та вивід;
* консольний інтерфейс.

## Зберігання даних

Інформація про задачі зберігається у текстовому файлі:

tasks.txt

Приклад запису:

Підготуватися до екзамену|5|0

Зробити лабораторну роботу|4|1

Купити продукти|2|0

де:

* 0 — активна задача;
* 1 — виконана задача.

## Структура проєкту

* `main.cpp` — вихідний код програми;
* `tasks.txt` — файл із даними;
* `README.md` — опис проєкту.
